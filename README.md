安卓本地录音同步至腾讯云 COS 的规格与文档仓库；第一期产品说明见 **`docs/SPEC.md`**。

## 自用场景约定（与规格书一致）

- **谁用、怎么装**：仅本人使用，不向他人分发。日常通过本机构建安装（Android Studio 运行到设备，或自行 `adb install` 生成的 APK）。
- **设备**：不做多机型矩阵；验收以本人单机为准。
- **COS 与桶内对象**：使用单一测试桶即可；上传到桶里的测试对象可长期保留，第一期不要求配置生命周期或定期清理。
- **状态与通知**：同步成败、可重试等**只在应用内**体现（列表状态枚举、打开 App 时的队列巡检等），**不使用系统通知栏**做业务类提醒。若平台对后台任务强制要求前台服务等系统级通知，以实现为准并在下文「权限与平台」中说明。
